Expected files   : aff_f.c
Allowed functions: write
--------------------------------------------------------------------------------

Écrire un programme qui prend en paramètre une chaîne de caractères, et qui
affiche sur la sortie standard le premier caractère 'f' rencontré dans cette
chaîne, suivi de `\n`.
Si aucun 'f' n'est rencontré, le programme affiche 'f' suivi de `\n`.
Si le nombre de paramètres est différent de 1, le programme affiche 'f' suivi
de `\n`.
